# Musarde

Mobile-first museum companion app. Pre-visit route planning and in-gallery
vision + RAG flow over open museum collection data (The Met, Art Institute
of Chicago, J. Paul Getty Museum).

**Status:** scaffolding. Active build starts Week 1 (May 4, 2026).

## Stack

Next.js (App Router) · Tailwind · Vercel · Postgres + pgvector · OpenCLIP
for image embeddings · Claude for agent reasoning and generation.
